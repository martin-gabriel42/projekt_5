# Projekt Power BI - ENGETO

## Tento repozitář je 5. projekt do kurzu Datový analytik s Pythonem. Téma projektu jsou volby do poslanecké sněmovny parlamentu České Republiky v roce 2025.

Repozitář obsahuje zazipovanou složku Projek_5.zip. Tato složka po extrahování obsahuje:
- soubor doc.md - dokumentace projektu
- soubor Power BI.pbix - hlavní výstup projektu
- složku Zdroje dat - složka se zdrojovými daty pro projekt

Zdrojová data lze stáhnout zde: https://www.volby.cz/opendata/ps2025/ps2025_opendata_seznam.htm

Odkazy na stáhnutí:
- prskl: https://www.volby.cz/opendata/ps2025/csv/psrkl.csv
- pst4: https://www.volby.cz/opendata/ps2025/csv/pst4.zip
- pst4p: https://www.volby.cz/opendata/ps2025/csv/pst4p.zip
- cnumnuts: https://www.volby.cz/opendata/ps2025/csv/cnumnuts.csv

Soubor "cz GeoJSON": https://simplemaps.com/gis/country/cz#admin1

Konverze "cz GeoJSON" na "cz topoJSON": https://mapshaper.org/
