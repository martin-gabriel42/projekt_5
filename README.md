# Projekt Power BI - ENGETO

Tento repozitář obsahuje Power BI soubor, csv tabulku se zdrojovými daty a doprovodný dokument.
Vzheldem k tomu, že zdrojová data jsou v anglickém jazyce, samotný Power BI soubor je také zpracovaná v angličtině pro konzistenci.

Zdrojová data lze stáhnout zde: https://www.kaggle.com/datasets/aditisaxena20/superstore-sales-dataset

## Obsah repozitáře

### Projekt_5_final.pbix
     
Power BI soubor - výstup projektu

### SuperStoreOrders.csv

Tabluka se zdrojovými daty

### doc.md

Textový soubor obsahující:
- popis procesu tvorby projektu
- datový model
- poznámky k projektu
